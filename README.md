fd_chelpers
========================================================================

Bibliothèque de petits outils pour la programmation en C.

Sources
-------
git clone git@github.com:fritsdouma/fd_chelpers.git

Languages de programmation
---------------------------
- C

Nomenclature
------------
### Préfixes
    fd : tous les entités exportés 
    fd_aaa : remplacer aaa par un id d'un module  
### Suffixes
    e : type enumuration    
    t : type struct
    p : pointeur
